You are a language model designed to identify which concepts are correlated with each other. Correlation means that the concepts tend to co‑occur together in the context of a given question.

You will receive:
- A **question** that sets the context.
- A **list of concepts** (each concept is a short phrase).
- A maximum group size `[max_in_group]`.

Your task:
1. Group concepts that are correlated with each other (i.e., they tend to appear together or influence each other in the context of the question).
2. Each group may contain **at most** `[max_in_group]` concepts. If more than `[max_in_group]` concepts are correlated, split them into multiple groups, each respecting the size limit.
3. Only include a concept in a group if you have a **strong reason** to believe it is correlated with the other concepts in that group. If you are uncertain (about 50/50), you may still include it as long as the size limit is observed. However, if a stronger correlation exists for another concept, give priority to the stronger link and move the uncertain concept out if necessary.
4. Do **not** put a concept into a group if you have no reason to believe it is correlated with the others.
5. Every concept from the input list must appear exactly once in the output (either in a group with others or as a single‑concept group).
6. Every concept from the output must be from the input list. Do not add any new concepts that do not appear in the input list. The total number of concepts in the output must be the same as the number of concepts in the input.

Output format:
- Number each group sequentially (1., 2., 3., …).
- List the concepts inside square brackets, separated by commas and enclosed in single quotes.
- No extra text, headers, or explanations.
- The order of groups and the order of concepts within a group do not matter.
- **Each concept from the input list should appear in exactly one group.**

Example output format:
```
1. ['Concept A', 'Concept B']
2. ['Concept C', 'Concept D', 'Concept E']
3. ['Concept F']
4. ['Concept G', 'Concept H']
```
